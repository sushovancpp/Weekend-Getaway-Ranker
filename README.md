# Weekend Getaway Ranker

## Project Description
A data engineering project that recommends top weekend travel destinations in India
based on distance, ratings, popularity, and visit duration.

## How to Run
1. Keep `Top Indian Places to Visit.csv` in this folder
2. Install dependencies:
   pip install -r requirements.txt
3. Run:
   python weekend_ranker.py

## Technologies
- Python
- Pandas
- Scikit-learn
